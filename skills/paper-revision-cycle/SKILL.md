# Paper Revision Cycle

## Steps
1. Read `.codex/AGENTS.md` and `references/revision_flow.md`.
2. Ask the user to paste reviewer comments as-is.
3. Produce a Revision Plan (PROPOSE) with IDs.
4. After approval, propose diffs for manuscript and rebuttal files.

## Rules
- Do not apply changes without approval.
- Keep all changes traceable by diff.
